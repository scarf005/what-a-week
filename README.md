# What A Week, Huh?

![](/public/kanna.png)

[original artwork by Maven](https://www.pixiv.net/artworks/115651167)

## Running

You need to have Deno v2.0.0 or later installed to run this repo.

Start a dev server:

```
$ deno task dev
```

## Deploy

Build production assets:

```
$ deno task build
```
